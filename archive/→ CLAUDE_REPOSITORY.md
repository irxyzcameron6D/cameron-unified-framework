## What this repository was

The Claude verification repository was built using Claude (Anthropic)
as a computational verification tool. Every claim in the repository
links to a calculation that produces a number. The number could have
come out wrong. When it did, that is documented.

The repository served as the "spine" of the framework development —
checking every equation from the Gemini synthesis for dimensional
consistency, computing specific numerical predictions, and identifying
where claims were postulates rather than derivations.

---

## What it produced

- Python Monte Carlo scripts for nuclear van der Waals binding (8–14σ)
- Numerical verification of the LMC orbital test (0.3% agreement)
- The perihelion advance derivation (3:2:1 ratio, 43.00 arcsec/cy)
- The relativistic beam deflection prediction table
- The hull temperature speedometer formula
- The spin and Pauli exclusion derivations from PMV bivector geometry
- Documentation of six open gaps in `disputes/what_doesnt_work.md`

## The correction history

Eleven errors were identified in the Gemini synthesis across six
iterations. The full correction history is in `COMPARISON.md` in
this central repository.

Score: 0/11 correct in Gemini V1 → 11/11 correct in the final version.

## How to use it

The central repository (`cameron-unified-framework`) contains the
final verified versions of all files from this repository. The
original repository is preserved as a historical record showing
the development process.

To reproduce any calculation from the original repository, use the
scripts in `montecarlo/` and `predictions/` in the central repository.
They are identical to the originals.
